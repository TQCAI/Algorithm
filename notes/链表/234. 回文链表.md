[234. 回文链表](https://leetcode-cn.com/problems/palindrome-linked-list/)
快慢指针 + 翻转链表
```python
class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        pre = None
        slow = head
        fast = head
        while fast is not None and fast.next is not None:
            tmp = slow.next
            slow.next = pre
            pre = slow
            fast = fast.next.next
            slow = tmp
```
