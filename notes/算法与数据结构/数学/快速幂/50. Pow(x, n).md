[50. Pow(x, n)](https://leetcode-cn.com/problems/powx-n/)

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n == 0:
            return 1
        if n < 0:
            x = 1 / x
            n = -n
        res = 1
        base = x
        while n:
            if n & 1:
                res *= base
            base *= base # 错误写成 *= x
            n >>= 1
        return res
```
